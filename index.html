recent web projects

NASYNTH

Our final Bootcamp project utilizes APIs and Javascript libraries to create an app for education, relaxation, and, my personal touch: generative soundscapes. NASYNTH provides the user full navigation of the 5-year archive NASA Astronomy Picture of the Day, giving you beautiful views and detailed explanations in desktop or mobile views. The bottom tray holds a 4-oscillator ambient soundscape generator, built with Tone.js and sequenced with some probability-based algorithm, built around Tonal.js scales. Users can save their “patches” and load up those made by anyone else. We hope NASYNTH can serve as a tool to casually explore sound and space! Huge thank you Daniel Bouchard and Dan Tramte for your inspirational tutorials on Tone and browser-based sound generation.

Tools used: Angular/Web Audio API/Tone.JS/HTML/CSS/Typescript

*ngFlix

To demonstrate our understanding of Angular and displaying data from Web APIs, we developed a platform for searching, filtering, and saving movies to a watch another time. Users can search a massive database of films, view additional movie details by clicking on the tiles, filter and sort by multiple parameters, and store films in a separate watchlist, indicated by an icon on each tile. We focused on a clean, mobile-first, desktop-distinct interface with smooth transitions between windows and functions. We plan to expand on the app by adding watch links for any films that are available to stream for free (legally).

Tools used: Angular/TMDB API/HTML/CSS/Javascript

boutique gan

For our first project in the GC Bootcamp, we decided to build a Point-Of-Sale system for a conceptual streetwear storefront. The shoes for sale are developed from 3 different base images, all "remixed" using the Artbreeder General Adversarial Network, or GAN. The store features a full functioning cart and checkout system, custom CSS animations, and scales well on mobile devices, though the ideal execution would be on an ipad in the middle of an unstaffed store. Customers would self-checkout and use the honor system...we are still working out the details there.

Tools used: HTML/CSS/Javascript

for the last couple of years, I have directed a late-night audio-visual event called Equalizer. We feature some of Detroit's best budding sonic talent and techno greats, matched with a state-of-the-art sound system and live-performed visuals. As an extension of the party, I curate the Equalizer Detroit radio show on UK-based Threads Radio, featuring recordings of my inspirational artists in the area as well as my own captured live sets.
